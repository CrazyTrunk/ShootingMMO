## Description

Một Game bắn súng hướng đến nhiều người chơi, max 6 players với ba chế độ đấu đơn, đấu đội và sống sót, có thể nhặt vũ khí để thêm đạn, có 3 loại súng mỗi loại sẽ gây damage khác nhau.
